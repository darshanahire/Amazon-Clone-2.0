
# Amazon Clone Application

A full-stack E-Commerce web application created using MERN Stack.

## Features of App
Following are the important fetures of App : 
<ul>
<li>Add Item to Cart</li>
<li>Like Items</li>
<li>Track Order</li>
<li>Cancel Order</li>
<li>Search Items</li>
</ul>
